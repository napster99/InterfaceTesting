# InterfaceTesting
Ajax接口性能评测工具

    | -- config.js      // ajax接口配置文件  
    | -- node_moudules  // Node所依赖的模块、包 express、mongodb 等  
    | -- app.js         // 程序执行入口  
    | -- package.json   // 项目info&npm包管理文件  
    | -- t.js           // 打印日志文件  

配置参数说明
  
  
* url>ajax请求地址
* count>发起请求次数
* query>请求参数
